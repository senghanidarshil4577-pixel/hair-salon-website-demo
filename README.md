# hair-salon-website-demo
Modern Barber Shop Website with Booking System and WhatsApp Integration

This is a modern and responsive barber shop website designed for local grooming businesses.

The website includes a complete appointment booking system where users can select services, choose a date, and submit their details.

It also features WhatsApp integration for instant customer communication, along with Google Maps for location visibility.

Key features:
• Professional barber-style UI design  
• Service showcase (Haircut, Beard, Fade)  
• Appointment booking form  
• Email notification system (via mailto)  
• WhatsApp integration  
• Google Maps location embed  
• Fully responsive design  

This project is ideal for barbershops, salons, and local service businesses looking to establish an online presence and generate leads.
